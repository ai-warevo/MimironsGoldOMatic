Acting as **[Senior Architect]**.

## Goal
Issue final production launch recommendation using completed gate evidence and residual risk analysis.

## Read first

## Execute
1. Verify each gate criterion has explicit pass/fail evidence.
2. Summarize residual risks with impact and accepted mitigations.
3. Record final decision (`Go` or `No-Go`) and sign-off metadata.
4. Define first 72-hour post-launch monitoring actions.

## Acceptance criteria
- Decision is evidence-based, auditable, and owner-signed.
- Residual risk posture is explicit.
- Immediate post-decision action plan is included.

## Output
